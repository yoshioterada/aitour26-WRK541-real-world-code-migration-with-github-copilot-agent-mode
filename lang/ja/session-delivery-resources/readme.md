## 🚀 はじめに

ワークショップは複数セクションで構成され、16 枚のスライド、1 つのデモ、2 つのハンズオンラボがあります。

### 🕐 タイミング

| 時間 | 説明 |
| -------------- | ------------- |
| 0:00 - 5:00 | 導入と概要 |
| 5:00 - 70:00 | セッション手順 |
| 70:00 - 75:00 | まとめ & Q&A |

### 🏋️ 準備

このワークショップに事前準備は不要です。参加者はセットアップなしで参加でき、必要な資料と手順はセッション中に提供されます。

### 🖥️ デモ

事前に構成済みのデモは不要です。このセッションは講師主導で進行し、ハンズオンラボを含みます。デモは当日ライブで実施し、事前セットアップは必要ありません。
