## 🎯 完成例を参照するタイミング

### ✅ **適切なタイミング**

1. **行き詰まったとき**
   - 10分以上デバッグしても解決しない
   - エラーメッセージがヒントにならない
   - 正常系と比較したい

2. **パターンを理解したいとき**
   - JSON デシリアライズの構造は？
   - エンドポイントの整理方法は？
   - C# でのモデル定義例を見たい

3. **コード比較したいとき**
   - 動いているが別案を見たい
   - ベストプラクティスに沿っているか確認したい
   - 別アプローチを学びたい

4. **時間が限られているとき**
   - 進捗が遅れている、追いつきたい
   - ゴールを把握するために結果を見たい
   - 終了間近で全体像を確認したい

### ⚠️ **避けるべきタイミング**

1. **自力で試す前**
   - 理解せずコピペしない
   - まず自分で挑戦する
   - 回答を見る前に Copilot を使う

2. **学習中**
   - つまずきも学びの一部
   - 失敗から学ぶ
   - 問題解決がスキルになる
