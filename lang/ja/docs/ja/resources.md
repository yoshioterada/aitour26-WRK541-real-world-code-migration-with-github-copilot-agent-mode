## ローカル環境の前提条件

GitHub Codespaces でなくローカルでワークショップを行う場合、以下をインストール済みであることを確認してください（インストール済みと仮定して演習を進めます）。

- VS Code で GitHub Copilot を有効にした GitHub アカウント
- Git（リポジトリクローン用）
- Visual Studio Code
- Python 3.12 と `pip`（`src/python-app/requirements.txt` のインストール用）
- .NET 10 SDK（C# Minimal API 用）
- 任意: VS Code の REST Client 拡張（エンドポイントテスト用）

詳細なインストールと検証手順は、リポジトリの [Local Development Prerequisites](../../PREREQUISITES.md)
を参照してください。
