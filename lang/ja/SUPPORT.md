# サポート

## Issue 登録とヘルプの受け方

このプロジェクトでは GitHub Issue でバグや機能要望を管理しています。重複を避けるため、まず既存の Issue を検索してください。新規の場合は、バグまたは機能要望を新しい Issue として登録してください。

このプロジェクトの利用方法に関する質問やヘルプについては、**リポジトリ管理者の方は、サポート方法をここに追記してください（例: Stack Overflow のタグや他のチャネル、連絡先など）**。
