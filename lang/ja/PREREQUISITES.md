# 前提条件 (PREREQUISITES)

このプレースホルダーファイルは、ワークショップをローカルで実行するために必要な前提条件を示します。

- Git
- GitHub Copilot が有効な VS Code
- Python 3.12
- .NET 10 SDK
- .NET 10 ランタイム（ビルド済みバイナリを実行する場合）

クイック検証手順:

1. `git --version`
2. `code --version`
3. `python --version`
4. `dotnet --info`

より詳細なチェックリストが必要な場合は、このファイルを更新し、具体的なバージョンやダウンロードリンクを追記してください。
